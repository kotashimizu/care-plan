# 改善方針まとめ

## 最終更新: 2025年1月19日

## 核心的な改善ポイント

### 🔴 最優先事項：言い回しの修正

**問題の本質**
- 永山さん：「内容は良かったが、言い回しの部分だけが問題」
- 指導対象となる重要事項

**具体的な修正内容**
```
❌ 現在の表現例：
「就労能力向上の具体的で測定可能な目標」
「日常生活技能向上の目標」

✅ 必要な表現例：
「毎日通所したい」
「自分で身支度ができるようになりたい」
「友達と楽しく話せるようになりたい」
```

### 意思決定支援への対応（2024年4月〜）

**背景**
- 福祉制度の新しい方向性として「意思決定支援」が重視
- 本人の意思と役割を明確にすることが必須要件

**実装への影響**
1. 到達目標 → 本人の願望として表現
2. 本人の役割 → 本人が主体的に取り組む内容
3. 支援内容 → 支援者が本人の意思を支える形で記載

## プロンプト修正の具体例

### 修正前のプロンプト
```javascript
"objective": "就労能力向上の具体的で測定可能な目標を80文字程度で設定してください。"
```

### 修正後のプロンプト
```javascript
"objective": "本人が希望する内容を『〜したい』『〜できるようになりたい』という形式で、本人の言葉として80文字程度で記載してください。"
```

### 支援内容の修正
```javascript
"supportContent": "本人の希望を実現するために、支援者が『〜を支援します』『〜のサポートを行います』という形式で、具体的な支援内容を80文字程度で記載してください。"
```

## B型事業所特化の強化

### 現在の強み
- B型事業所特化指針は既に実装済み
- 生活リズム、作業継続性、本人ペースの尊重などが含まれている

### さらなる強化ポイント
1. **作業訓練の段階的目標**
   - 短時間から始める
   - 徐々に時間を延ばす
   - 本人の体調に配慮

2. **生活面のサポート**
   - 通所の安定
   - 生活リズムの確立
   - 社会参加の促進

## 書式の整理

### 必須5項目（下半分）
1. **到達目標** - 本人の願望形式
2. **本人の役割** - 本人が取り組むこと
3. **支援内容** - 支援者が行うサポート
4. **期間** - 支援期間・頻度
5. **優先度** - 1, 2, 3の数字

### 上半分
- 現状維持でOK
- サービス等利用計画書との連携は将来的な課題
